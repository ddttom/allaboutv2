# Scripts

Core JavaScript files for EDS site functionality.

## Key Files

| File | Purpose |
|------|---------|
| `aem.js` | AEM/EDS core utilities |
| `scripts.js` | Site initialization |
| `delayed.js` | Lazy-loaded scripts |

## Usage

Scripts are automatically loaded by EDS.
Blocks import utilities from aem.js.
