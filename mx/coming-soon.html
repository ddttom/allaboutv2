What's happening right now

A river cruise costs £2,000. The AI agent reads it as £203,000. A support article explains the fix. The agent says no fix exists.

January 2026 changed everything. Amazon launched Alexa+ (browser agent). Microsoft launched Copilot Checkout. Google launched UCP. Anthropic launched Claude Cowork. The big players are building tools for AI to act on web pages — to press buttons and fill forms.

But nobody is making the content itself machine-readable. That's the gap.

What MX is

Machine Experience (MX) is structured metadata alongside your existing content. The metadata is for machines. The prose is for humans. Same content, both audiences served.

No APIs. No code changes. Just text files that any AI agent can read — from GPT-4 to a 50M parameter model running locally.

We call it the companion web. Your website stays exactly as it is. MX adds understanding.
