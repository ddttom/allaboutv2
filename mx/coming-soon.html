The Problem

AI agents are reading the web and getting it wrong. A river cruise costs two thousand pounds. The agent reads two hundred and three thousand. A support article explains the fix. The agent says no fix exists. Every wrong answer wastes electricity, loses sales, erodes trust.

The industry is building tools for agents to act on web pages — to press buttons and fill forms. But nobody is making the content itself machine-readable. Actions without understanding.

The Solution

Machine Experience (MX) is a metadata layer that sits alongside your existing content. The metadata is for machines. The prose is for people. Same content, both audiences served. No APIs. No code. Just structured text files that any AI agent on earth can read.

We call it the companion web — a machine-readable layer alongside the human web. Your website stays exactly as it is. MX adds understanding.
