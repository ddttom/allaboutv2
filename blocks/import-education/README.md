# import-education

This block represents the Education section of a LinkedIn profile, displaying a user's academic history.

## Usage

Use this block to showcase educational qualifications in chronological order.

## Authoring

In your Google Docs or Microsoft Word document, create a table with the following structure:

| import-education |
| :---- |
| [School Name] |
| [Degree] |
| [Duration] |

Repeat the rows for each education entry.

## Styling

The block uses custom CSS classes for styling. You can customize the appearance by modifying the `import-education.css` file.

## Behavior

The block creates a list of educational experiences, each displaying the school name, degree, and duration.

## Accessibility

The block uses semantic HTML elements to ensure proper structure and readability for screen readers.
