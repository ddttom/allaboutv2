# allabout.network Website

## Purpose and Role

Personal website repository for allabout.network. Contains all website code, published blog HTML, demo sites (LPC reference, Reginald, CogNovaMX), and static assets.

This is a **personal mount** — another business deploying MX would mount their own website repository at this mount point.

## Mount Type

**Personal** — swappable per business. Tom's allaboutv2 is his website. Scott's business would mount a different repo here.
