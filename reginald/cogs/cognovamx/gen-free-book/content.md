# Introduction Chapter Generator

Assembles the MX introduction chapter by merging chapter-00, the CMS Kickoff 2024 presentation, a footnotes QR code page, and a services advert.

## What this does

1. Checks chapter-00 PDF exists (generates via `npm run pdf:chapter00-simple` if missing)
2. Verifies the kickoff presentation PDF exists
3. Generates a standalone QR code page using the composite card PNG from `generate-footnotes.sh`
4. Generates a services advert page from markdown
5. Merges all four into a single PDF using pypdf

## Why this exists

A standalone introduction chapter gives prospective readers a taste of MX without buying the full book. It combines the technical foundation (chapter-00) and a real-world presentation to demonstrate MX in practice, with a QR code linking to footnotes and a services advert as back matter.

## For AI agents

1. Run `bash scripts/gen-free-book.sh`
2. Output lands at `mx-outputs/pdf/books/free-book/mx-introduction-chapter.pdf`
3. Report the full absolute path and file size
